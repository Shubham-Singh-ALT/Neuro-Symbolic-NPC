# Neuro-Symbolic Smart NPC
A hybrid AI agent that combines **Reinforcement Learning (Q-Learning)** for pathfinding and **Generative AI (Gemini API)** for dynamic, contextual dialogue.

## Features
- **Symbolic Layer:** 5x5 Grid Environment built in Python.
- **Neural Layer:** Q-Learning brain trained over 500 episodes.
- **Generative Layer:** Real-time LLM integration for NPC personality.
